## 基礎成本計算方式

```
基礎成本 = 固定費 + 畫布印製費 + 框架費
（不含顏料，顏料成本納入難易度係數）
```

**固定費：NT$25**
| 項目 | 費用 |
|------|------|
| 畫具（筆刷等）| NT$5 |
| 包裝 | NT$20 |

**畫布印製費**
```
實際印製尺寸 = (畫布寬 + 10cm) × (畫布高 + 10cm)（四周各加 5cm 裝訂用）
印製費 = ceil(實際面積 ÷ 900) × NT$45
```
> 印刷廠最低起印 NT$200，但每次批量印刷皆超過此門檻，不影響單件成本。

**框架費**
```
框架費 = 畫布寬 + 畫布高（NT$）
邏輯：實木框以周長計費，單價 NT$0.5/cm
```

## 定價公式

```
售價 = (基礎成本 + 顏料費) × (1 + 細緻度加成率)
顏料費 = 色數 × NT$1
```

**難易度 → 色數**

顏料費依 `production_job.num_colors_used`（演算法實際使用色數）計算，非難易度預設上限。

| 難易度 | 色數上限（參考）| 顏料費計算 |
|------|------|------|
| 入門 | 18色 | num_colors_used × NT$1 |
| 初級 | 24色 | num_colors_used × NT$1 |
| 中級 | 35色 | num_colors_used × NT$1 |
| 進階 | 50色 | num_colors_used × NT$1 |

> 同難易度的不同商品因實際色數不同，售價可能略有差異。系統在建立變體時自動帶入公式計算值，管理員可手動調整。

**細緻度加成率**

| 細緻度 | 加成率 | 倍數 |
|------|------|------|
| 粗糙 | 60% | ×1.60 |
| 標準 | 75% | ×1.75 |
| 細緻 | 80% | ×1.80 |
| 高級 | 85% | ×1.85 |

> 錨點：30×40 入門 粗糙 = (NT$230 + NT$18) × 1.60 = **NT$397**

## 客製服務報價

### 客製照片（custom_photo）

```
建議報價 = 基礎價（尺寸 × 難易度）+ 勾選的加費項目
管理員可手動調整最終 quoted_price
```

- 細緻度由管理員根據照片品質決定，不影響定價
- SAM 遮罩成本已包含在基礎定價內，不另計
- 所有金額可在後台調整，不需改程式碼

**後台報價流程**

1. 系統帶入基礎價（尺寸 × 難易度表查詢）
2. 管理員勾選適用的加費項目
3. 系統自動加總顯示建議報價
4. 管理員確認或手動覆蓋 → 填入 `quoted_price`

| 欄位 | 說明 |
|------|------|
| id | 主鍵 |

| label | 顯示名稱（如「2人」、「寵物」）|
| amount | 加費金額 |
| is_active | 是否啟用 |

### 客製規格（custom_spec）

```
建議報價 = 公式價格（同目錄商品，不加任何服務費）
管理員確認即可，可手動微調
```

**報價明細範例**

| 項目 | 說明 | 金額 |
|------|------|------|
| 基礎成本 | 40×50 | NT$295 |
| 顏料費 | 中級 35色 × NT$1 | NT$35 |

| **最終 quoted_price** | | **NT$578** |
