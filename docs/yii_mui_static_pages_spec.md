## 第一頁:什麼是數字油畫

### 頁面定位

讓從來沒聽過「數字油畫」的人,在 30 秒內理解三件事:
1. 這是什麼
2. 我會不會畫
3. 適不適合我

**情緒順序**:無門檻 → 療癒 → 送禮(三層遞進)

### 頁面架構

```
┌─────────────────────────────────────┐
│  Section 1:開場(水彩風主視覺)     │
├─────────────────────────────────────┤
│  Section 2:四步驟說明(線條 icon)│
├─────────────────────────────────────┤
│  Section 3:療癒氛圍區             │
├─────────────────────────────────────┤
│  Section 4:成品展示               │
├─────────────────────────────────────┤
│  Section 5:CTA(現成款 + 客製並列)│
└─────────────────────────────────────┘
```

### Section 1:開場

**視覺**:水彩風主視覺
- 元素:畫架 + 半完成的畫(色號可見)+ 茶杯 + 葉子 + 葉片暈染
- 配色:sage 綠 + 暖橘小點綴 + cream 主調
- 方向:橫向,呈現「桌面氛圍」

**標籤(視覺上方)**:
```
YII.MUI ・ 數字油畫
```
小字、letter-spacing 4px、`#8a8a82`

**主標題**(襯線體 36px,黑灰色 `#50504B`):
> 一個人、一杯茶、一下午
> 一幅屬於你的畫

**副標**(無襯線 14px,`#6e6e68`):
> 數字油畫 kit 把畫畫拆解成「對應數字、塗上顏色」
> 讓沒接觸過畫畫的你,也能完成一幅可以掛起來的作品。

### Section 2:四步驟說明

**標籤**:
```
HOW IT WORKS
```

**Section 標題**(襯線 26px):
> 四個步驟,慢慢來就好

**四個 step**(橫排,每個 step 用 110px 圓形 + 線條 icon):

| Step | 標題 | 內文 | Icon 內容 |
|---|---|---|---|
| 01 | 打開 kit | 畫布、顏料、畫筆、色號表,一次到齊。 | 線條描繪「打開的盒子」 |
| 02 | 沿著色號 | 畫布上每個區塊都有對應的數字。 | 線條描繪「畫布 + 數字 1-8」 |
| 03 | 一筆一筆 | 不用思考,慢慢上色就好。 | 線條描繪「畫筆 + 顏料點」 |
| 04 | 掛起來 | 完成的那一刻,它就屬於你了。 | 線條描繪「框 + 牆上釘子」 |

**Step 編號樣式**:11px、letter-spacing 2px、暖橘色 `#C68B5A`、上方標示「STEP 01-04」

### Section 3:療癒氛圍區

**視覺**:水彩風中型氛圍圖(可以是「茶杯側面 + 翻開的書 + 一支畫筆」)

**Section 文案**(置中,內縮排版):
> 一個週末。
> 一首喜歡的歌。
> 一杯咖啡。
> 一幅慢慢誕生的畫。
>
> 這就是 yii.mui 想送你的下午。

**字級**:15px,行高 1.9,`#6e6e68`

### Section 4:成品展示

**視覺**:**暫時用「線稿示意 + 預期成品圖」**(等鎮店之寶完成後換真照)
- 上排:3 張線稿縮圖
- 下排:3 張對應的「預期成品」示意

**短文案**:
> 完成後的它,可以掛在你最常待的角落。

### Section 5:CTA

**並列兩個按鈕**(視覺上等重)

| 左卡片 | 右卡片 |
|---|---|
| **現成款** | **客製服務** |
| 選一幅你喜歡的,立刻開始 | 把照片變成畫,專屬於你 |
| `[ 看現成款 → ]` | `[ 聊客製 → ]` |

按鈕樣式:
- 細線邊框 1px solid `#A89880`

## 第二頁:客製化流程

### 頁面定位

讓客人從「好像可以客製」變成「我知道怎麼客製、想試試看」。

**主要解決三個疑慮**:
1. 我的照片可以嗎?
2. 流程會不會很麻煩?
3. 出來會跟我心目中一樣嗎?

### 頁面架構

```
┌─────────────────────────────────────┐
│  Section 1:開場(對話導向)         │
├─────────────────────────────────────┤
│  Section 2:四步驟流程              │
├─────────────────────────────────────┤
│  Section 3:照片指南(對照式)       │
├─────────────────────────────────────┤
│  Section 4:CTA「立即上傳照片」     │
└─────────────────────────────────────┘
```

### Section 1:開場

**視覺**:水彩風(主題:照片 → 線稿 → 完成品的「轉變過程」)
- 三個重疊的圓:左邊「照片」、中間「線稿」、右邊「彩色畫」
- 用水彩柔邊串聯三個階段

**主標題**:
> 有些照片,值得花一點時間慢慢畫

**副標**:
> 你不用會畫畫。
> 把那張對你有意義的照片交給我,
> 我幫你變成可以親手完成的畫。

### Section 2:四步驟流程

**標籤**:`HOW CUSTOM WORKS`

**Section 標題**:
> 四個步驟,把照片變成畫

**四個 step**(線條 icon,跟第一頁同一個視覺語言):

| Step | 標題 | 內文 |
|---|---|---|
| 01 | 選尺寸 + 上傳照片 | 選擇你想要的畫布尺寸,上傳一張對你有意義的照片。 |
| 02 | 我幫你轉成線稿 | 1-2 個工作天內,我會把照片轉成可以畫的線稿。 |
| 03 | 你確認、可微調 | 你會看到線稿,可以微調最多 3 次,直到你滿意。 |
| 04 | 印製寄出 | 你按下「確認製作」,我開始印製、配料、寄出。 |

**重要備註框**(在 Section 2 下方):
> ※ 按下「確認製作」後即進入印製階段,不可退換。
> 確認前若有疑問,可以無限次討論。

### Section 3:照片指南

**Section 標題**:
> 怎麼選一張適合的照片?

**視覺呈現**:左右對照式,每邊 3-4 張小示意圖 + 簡短說明

**左欄(綠色標題「✓ 適合」)**:
- 主體清楚(人/寵物/物件佔畫面 1/2 以上)
- 光線充足(自然光最好)
- 解析度夠(手機原檔即可)
- 背景單純(純色牆、單色布料最佳)

**右欄(灰色標題「✗ 不適合」)**:
- 逆光、背光(看不清主體)
- 過度模糊或晃動
- 從低解析度截圖、社群縮圖
- 過度複雜的背景

**示意圖風格**:極簡線條風、不上色、用線條輪廓示意。

### Section 4:CTA

**單一按鈕**(主要色強調,視覺重量比一般按鈕重):
```
[ 立即上傳照片 → ]
```

按鈕下方一行小字:
> 看不到自己想客製的圖?私訊我聊聊也可以。

## 第三頁:尺寸指南

### 頁面定位

讓客人在 1 分鐘內找到「適合自己的尺寸」。

**最大挑戰**:客人腦袋裡沒有「30×40」是多大的概念,要轉成生活化對照。

### 頁面架構

```
┌─────────────────────────────────────┐
│  Section 1:開場(極簡)            │
├─────────────────────────────────────┤
│  Section 2:首波 4 個尺寸           │
│  20×20 / 30×30 / 30×40 / 40×50    │
│  圖文對照(每個尺寸一張卡片)        │
├─────────────────────────────────────┤
│  Section 3:客製推薦尺寸            │
│  40×40 / 50×50 / 50×60            │
│  + CTA → 客製頁                    │
├─────────────────────────────────────┤
│  Section 4:折疊區「完整 16 種尺寸」│
│  含售價區間 + 超商取貨規則         │
└─────────────────────────────────────┘
```

### Section 1:開場

**短標題**:
> 該選多大?跟著畫面感覺走

**副標**:
> 從 20×20 的書桌小品,到 60×60 的客廳主視覺,
> 找到適合你空間的那一幅。

### Section 2:首波 4 個尺寸

**Section 標題**:
```
SIGNATURE SIZES
```

**視覺呈現**:每個尺寸一張橫向卡片,左圖右文

**圖**(畫法 C 變化版:乾淨方塊 + 水彩對照物):
- 場景:側視桌面
- 中央:sage 綠方塊代表「畫」,內標「30×30 cm」字樣
- 周圍對照物:茶杯、翻開的書、小盆栽,用水彩柔邊
- 桌面:淡米色橫條
- 對照物大小要對應「該尺寸的真實比例」

**文**(右欄):
| 元素 | 內容 |
|---|---|
| 尺寸標題 | 例:30 × 30 cm |
| 尺寸對照 | 例:≈ 一本雜誌大小 |
| 適合擺放 | 📍 書桌、床頭、玄關 |
| 適合題材 | 🎨 單一花朵、寵物頭像 |

**4 個尺寸的具體內容**:

#### 20×20 cm
- ≈ 一張 CD 大小
- 📍 書桌、床頭櫃、層架小角
- 🎨 迷你花卉、寵物大頭照
- 🕒 半天 ~ 一天可完成

#### 30×30 cm
- ≈ 一本雜誌大小
- 📍 書桌、玄關、茶几
- 🎨 單一花朵、單一動物
- 🕒 1 ~ 2 天

#### 30×40 cm(直幅)
- ≈ A3 紙
- 📍 沙發牆、臥室掛畫
- 🎨 風景、人物半身、寵物全身
- 🕒 1 週左右

#### 40×50 cm
- ≈ 筆電打開的螢幕
- 📍 客廳主視覺、玄關大牆
- 🎨 完整風景、家庭合照、紀念場景
- 🕒 2-3 週

### Section 3:客製推薦尺寸

**Section 標題**:
> 想做更大的作品?
> 這幾個尺寸最適合客製

**視覺呈現**:橫排 3 個小卡片(比 Section 2 簡化)

#### 40×40 cm(正方形)

#### 50×50 cm(正方形)
- 客製主流款

#### 50×60 cm(橫幅)
- 客製旗艦款
- 適合家庭橫照、情侶橫照、紀念場景

### Section 5:運費規則

**Section 標題**:`運費規則`

> 現成款:滿 NT$1000 免運
> 客製款:依實際運送方式收取

**品牌說明**:
> 客製款是 yii.mui 為你獨立準備的作品,

### Section 2-3:4 個分類 + 15 題

#### 關於產品(4 題)

**Q. 我從來沒畫過,會不會太難?**
A. 數字油畫不需要美術底子。每個區塊都有對應數字,只要照數字塗就好。建議從入門款的 20×20 或 30×30 開始,完成一幅後你會比想像中還有信心。

**Q. kit 裡面有什麼?需要再買畫框嗎?**
A. 每份 kit 包含:已繃好的畫布(已含基底框架)、配好色號的顏料、3 支不同粗細的畫筆、色號對照表、操作說明書。基本款不額外附畫框,但畫布本身已可掛牆。

**Q. 顏料會不會有毒?小孩或寵物碰到安全嗎?**
A. yii.mui 使用的是水性壓克力顏料,通過安全認證,不含有害物質。但仍建議避免幼兒誤食、寵物舔食。畫畫過程中保持通風即可。

**Q. 完成的畫可以保存多久?會褪色嗎?**
A. 壓克力顏料完全乾燥後具備不錯的耐久度,在正常室內環境下可保存 5-10 年以上。避免長時間日曬,可延緩褪色。

#### 關於下單(3 題)

**Q. 結帳怎麼付款?**
A. 目前提供銀行轉帳付款。下單後 24 小時內完成轉帳並回填末五碼,即可進入製作流程。

**Q. 多久會收到?**
A. 現成款下單後 5-10 個工作天製作 + 1-3 天配送。客製款轉檔 1-2 天 + 確認後 5-10 天製作 + 1-3 天配送。節慶旺季前下單建議多預留時間。

**Q. 可以送到指定地址當禮物嗎?**
A. 可以。下單時填收件人姓名、地址、電話即可,我們不會在包裝上標示金額。如需附上手寫卡片,下單備註欄留言告訴我。

#### 關於客製(5 題)

**Q. 什麼樣的照片適合做客製?**
A. 主體清楚、光線充足、背景單純的照片最適合。逆光、過度模糊、低解析度截圖會影響線稿品質。客製化流程頁面有完整的照片指南。

**Q. 客製多久收到?**
A. 上傳照片後 1-2 個工作天內,我會把線稿傳給你確認。確認後進入印製,約 5-10 個工作天 + 1-3 天配送。

**Q. 線稿可以修改幾次?**
A. 確認前最多可修改 3 次。每次修改我會根據你的回饋調整線稿,直到你滿意為止。

**Q. 可以指定換背景嗎?**
A. 可以。客製轉檔時可以告訴我你想要的背景樣式(純色、簡化背景、或保留原圖),我會在線稿階段幫你處理。

**Q. 可以做明星 / 偶像的肖像嗎?**
A. 不行。yii.mui 不接受公眾人物肖像、卡通角色、品牌商標等涉及肖像權與智慧財產權的客製。客製內容請以你自己的照片(寵物、家人、朋友、自己)為主。

#### 關於配送(3 題)

**Q. 大尺寸超商真的不能取嗎?**
A. 任一邊長超過 40cm 的尺寸建議改選宅配。雖然有些超商可以勉強寄送,但運送過程中包裹擠壓會影響畫布品質,所以 yii.mui 預設大尺寸採宅配。

**Q. 包裝會不會壓壞?**
A. 每份 kit 都用紙板加固 + 減塑緩衝包裝,正常運送下不會變形。如收到時包裝有明顯損傷,請拍照在 7 天內告知,我會處理。

**Q. 偏遠地區會加運費嗎?**
A. 台灣本島不加價。離島(澎湖、金門、馬祖)需另計運費,下單時系統會提示。
